           ***********************************************
           *            GEBCO One Minute Grid            *
           ***********************************************

This directory contains the GEBCO one minute gridded data set.
Documentation on this data set can be found in the file GridHelp.pdf.
Information concerning the format of the data set can be found in
the file FORMAT.TXT.

The data contents of files within this directory are as
follows:

README.TXT       This file
gridone.grd      The GEBCO one minute gridded data set
FORMAT.TXT       Data format definition
GridHelp.pdf     Documentation to accompany the GEBCO one minute grid
